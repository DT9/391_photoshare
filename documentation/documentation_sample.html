User Documentation (LAOP stack (Linux Apache OracleSQL PHP))

Installation

1. Download project.tgz and put it an appropriate folder you want your web document (eg. web_docs)

2. Extract the file in the folder using command: tar xzvf project.tgz

3. Go into the web_docs folder using command: cd web_docs

4. run setup.sh file with appropriate credentials: sh setup.sh username password

5. Navigate to mainpage.html http://consort.cs.ualberta.ca/~/391_photoshare/mainpage.html

User Manual

Main Page (login.html)

This is the main page (index.html) for the website. There are two options to go login with your username and password or go to the user documentation. You can go to these option by clicking the buttons labeled User Login and User Documentation.

Login Module

Once you have navigated to the User Login you will be presented with a form to input your username and password. Once you have entered your username and password click the login button. If your username and password are correct it should log you in to your login information.

Login Information (Login Module)

Depending on your class you will be presented different options for buttons which lead to all of the other modules on the radiology website:

Logout (Login Module)

This button is present to all classes on the top of the login information page. Clicking this button will simply logout the user.

Modify Account (Login Module)

This button is present to all classes on the top of the login information page under the logout button. This button navigates the user to another page where a user can modify their account.

This option is used to update the user name they are using. They can do this by filling in the form (shown above) with the name they wish to change to and click Update name.

Change Password

This button is present to all classes on the top of the login information page under the logout button. This button navigates the user to another page where a user can change their password.

This option is used to change the password they are using. They can do this by filling in the password they wish to change to in form (shown above) and clicking Update name.

User Management

Insert Person (User Management Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can add a person to the database.

On this page there is a form where an administrator has to entering inputs first name, last

name, name, address, and phone. Once the administrator has successfully done this they can click Create person. This person will then be added to the database.

Update Person (User Management Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can update a person in the database.

In this form it first asks you to enter a name to the person you want to update and press Find User. Once the person is successfully found it will open another form that is populated with the current person data. You can edit this information as you choose and then click Update User to update the person.

Insert User (User Management Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can add a user to the database.

On this page there is a form where an administrator has to entering inputs a persons name, username, password, and a class they will be under. Once the administrator is finished entering information they can click create user and they will be added to the database.

Update User (User Management Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can update a user in the database.

In this form it first asks you to enter a username to the user you want to update and press Find User. Once the user is successfully found it will open another form that is populated with the current user data. You can edit this information as you choose and then click Update User to update the user.

Insert group (User Management)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can add a group to the database.

On this page there is a form where an administrator has to entering inputs the name of the photo and group they want to add. Once the administrator is finished entering information they can click “Create group” and they will be added to the database.

Update group (User Management Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can update a family entry in the database.

On this page there is a form where an administrator has to entering inputs the name of the photo and group they want to update and click “Search group. Once it has searched successfully it will open up another form asking the administrator to enter a photo name and group name they want to update the group entry with.

OLAP report (Report Generating Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can generate a report based on a diagnostic.

On this page there is a form where an administrator has to entering inputs the olap of report they want to generate and click “search”. Once it has searched successfully it will generate a report.

Insert Image (Uploading Module)

This button is present to only users with the class image present. This button navigates the administrator to another page where they can insert a image .

On this page there is a form where a image has to enter the inputs name, date, subject, place, description, and any pictures the image chooses to upload. To upload image click the browse button and a file manager should open. Within that you can select an image or multiple images holding (CTRL) and select what images you want. Once the image has uploaded what she wants she can click “Create Image” and the will be created.

Search Image (Search Module)

This button is present to all users. This button navigates the user to another page where they can search for Images

On this page there is a form where users can input keywords they want to be searched within Images. Also, a user can choose time periods to search for Images in between along with listing the s by Most Recent First/Most Recent Last. Once the user has parameter of search they want they can click “Search” and it will create a list based upon results of your search.

Data Analysis (Data Analysis Module)

This button is present to only users with the admin present. This button navigates the administrator to another page where they can do data analysis.

On this page there is a form where users can input whether or not they want to count the number of images a certain groups have owner name, subject, and Time (including weekly,monthly, and yearly). Once the administrator has selected what groups they want the data analysis on they can click “OLAP report and a table will be generated based upon the groups selected.
